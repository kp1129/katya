Conway's Game of Life

Tech

This app was built using React and bootstrapped with create-react-app. Styling is done with plain CSS. The app is deployed on Vercel (previously Zeit).

About the app

This project is my implementation of a computer science classic, John Conway's "Game of Life." The program simulates cellular automaton and allows the user to observe how their initial population (generation 0) will evolve over time based on the rules of the game. Click here to learn more about why this game is still relevant today (link to an article on Frontiers in Cellular and Infection Microbiology).

A user can set the initial population by clicking on the individual squares or by using the random population button that generates a random layout for the grid. The user can then step through each following generation of the initial population using the "see next generation" button. Alternatively, instead of viewing each generation one by one, the user can launch the simulation, and select their preferred simulation speed (slow, average, or fast). While the simulation is running, the user cannot change the state of any cells on the grid, but they may stop the simulation, make adjustments to the grid, and resume.

The app interface also includes the rules of Conway's Game of Life for those unfamiliar with it.

Learning process

Something new I learned during this project is React's useCallback hook. Previously, I'd only used useState and useEffect, as well as my own custom hooks. It was also fun making this app mobile and tablet-responsive using simple media queries in CSS. My favorite part about this app is that I got to build a scientific simulation. I strongly believe that just about every industry today would advance in leaps and bounds if they had access to better software tools, and this project is just the beginning of what I hope to contribute to the scientific community.
